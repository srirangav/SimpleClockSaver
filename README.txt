SimpleClock.saver v1.0.4
By Sriranga Veeraraghavan <ranga@calalum.org>

SimpleClock.saver is a simple Swift-based digital clock screensaver
for MacOSX 

To install:

    Click on SimpleClock.saver and follow the instructions in the 
    System Preferences.  

Supported MacOSX versions: 

    10.11+

License:

    Please see LICENSE.txt

History:

    v. 1.0.0 (03/22/2021) - Initial Release
    v. 1.0.1 (04/22/2021) - Add support for stardates
    v. 1.0.2 (05/02/2021) - Add support for timezones
    v. 1.0.3 (05/17/2021) - Add preference for display on main screen 
                            only
    v. 1.0.4 (05/21/2021) - Add support for TOS style stardates

Known Issues:

    On Big Sur (11.x), to access the settings after installation, 
    it may be necessary to quit System Preferences and re-launch it.
